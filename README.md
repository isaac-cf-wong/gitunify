## Overview

**gitunify** is a Python package that provides a unified interface to manage different Git platforms such as
GitHub, GitLab, Gitea, and more.
It abstracts the differences between various Git hosting services, allowing developers to interact with repositories,
issues, pull requests, and other Git-related features using a consistent API.

Whether you're working with multiple Git platforms or need to migrate between them,
gitunify simplifies the process by offering a single, easy-to-use interface.
